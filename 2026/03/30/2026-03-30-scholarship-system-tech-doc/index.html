数字化奖学金评定系统 技术文档

1. 项目概述

项目背景

传统奖学金评定流程依赖纸质材料，存在审批效率低、材料核实困难、流程不透明等问题。本项目构建数字化奖学金评定系统，实现从申请、审核到公示的全流程线上化，并集成OCR证件识别技术提升材料核验效率。

项目目标

  • 实现奖学金申请全流程线上化
  • 多级审批工作流自动化
  • OCR证件识别自动核验材料
  • 三端协同（Web管理端 + 移动端 + 后端API）

项目成果

  • 🏆 省级大创项目
  • 🏆 中国计算机设计大赛国家级三等奖
  • 服务 千名学生，已稳定运行
  • 多级审批工作流完整实现
  • PaddleOCR证件识别集成

4. 核心功能模块

4.1 学生申请模块（UniApp移动端）

  • 在线申请：填写奖学金申请表，上传证明材料
  • 材料上传：拍照/相册上传证件照片
  • 进度查询：实时查看审批进度
  • 消息通知：审批结果推送通知
  • 历史记录：查看历次申请记录

4.2 多级审批工作流模块

  • 角色权限：学生、班主任、院系管理员、校级管理员
  • 审批记录：完整审计日志
  • 批量审批：支持批量通过/驳回
  • 超时提醒：审批超时自动提醒

4.3 OCR证件识别模块

  • 证件类型：身份证、学生证、获奖证书、成绩单
  • PaddleDetection：定位证件区域，裁剪关键信息
  • PaddleOCR：识别证件文字内容
  • 自动填充：OCR结果自动填充申请表单
  • 人工校验：OCR结果可人工修改确认

4.4 管理端模块（Vue Web）

  • 奖学金管理：奖项设置、名额分配、评审规则配置
  • 学生管理：信息维护、班级管理
  • 审批管理：审批队列、审批操作
  • 数据统计：申请统计、通过率分析、可视化报表
  • 公示管理：结果公示发布

7. 部署方案

  • 数据库：MySQL + Redis
